# basePath

`basePath` is an optional string parameter that specifies the folder where new files should be created when clicking on a heatmap box that doesn't have a corresponding file.

If `basePath` is set, and you click on a day without a file, the plugin will propose to create a new file at `basePath/YYYY-MM-DD.md`.

## Example

```dataviewjs
const trackerData = {
    entries: [],
    basePath: "daily notes", // New files will be created in "daily notes" folder
    heatmapTitle: "Heatmap with basePath",
}

renderHeatmapTracker(this.container, trackerData);
```
