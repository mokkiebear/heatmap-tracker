# basePath

> 📖 Full parameter reference: [README → `basePath`](https://github.com/mokkiebear/heatmap-tracker#basepath)

## Example

```dataviewjs
const trackerData = {
    entries: [],
    basePath: "daily notes", // New files will be created in "daily notes" folder
    heatmapTitle: "Heatmap with basePath",
}

renderHeatmapTracker(this.container, trackerData);
```
