# colorScheme

`colorScheme` allows you to define the colors used in the heatmap. You can use a predefined palette or specify custom colors.

## Properties

- `paletteName` (optional): The name of a predefined palette (e.g., "default", "github", "winter", "warm").
- `customColors` (optional): An array of hex color strings to use as the color scale.

## Example 1: Using a predefined palette

```dataviewjs
const trackerData = {
    entries: [],
    colorScheme: {
        paletteName: "winter"
    },
    heatmapTitle: "Winter Palette",
}

renderHeatmapTracker(this.container, trackerData);
```

## Example 2: Using custom colors

```dataviewjs
const trackerData = {
    entries: [],
    colorScheme: {
        customColors: [
            "#ebedf0", // 0 intensity (background)
            "#9be9a8",
            "#40c463",
            "#30a14e",
            "#216e39"  // Max intensity
        ]
    },
    heatmapTitle: "Custom Colors (GitHub style)",
}

renderHeatmapTracker(this.container, trackerData);
```
