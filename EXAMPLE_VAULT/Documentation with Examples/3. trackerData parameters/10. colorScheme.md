# colorScheme

> 📖 Full parameter reference: [README → `colorScheme`](https://github.com/mokkiebear/heatmap-tracker#colorscheme)

## Example 1: Using a predefined palette

```dataviewjs
const trackerData = {
    entries: [],
    colorScheme: {
        paletteName: "winter"
    },
    heatmapTitle: "Winter Palette",
}

renderHeatmapTracker(this.container, trackerData);
```

## Example 2: Using custom colors

```dataviewjs
const trackerData = {
    entries: [],
    colorScheme: {
        customColors: [
            "#ebedf0", // 0 intensity (background)
            "#9be9a8",
            "#40c463",
            "#30a14e",
            "#216e39"  // Max intensity
        ]
    },
    heatmapTitle: "Custom Colors (GitHub style)",
}

renderHeatmapTracker(this.container, trackerData);
```
