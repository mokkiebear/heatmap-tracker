# intensityConfig

> 📖 Full parameter reference: [README → `intensityConfig`](https://github.com/mokkiebear/heatmap-tracker#intensityconfig)

## Example

```dataviewjs
const trackerData = {
    entries: [
        { date: "2025-01-01", intensity: 10 },
        { date: "2025-01-02", intensity: 1 },
    ],
    intensityConfig: {
        scaleStart: 1,
        scaleEnd: 10,
        excludeFalsy: true,
        defaultIntensity: 5,
        showOutOfRange: true
    },
    heatmapTitle: "Heatmap with intensityConfig",
}

renderHeatmapTracker(this.container, trackerData);
```
