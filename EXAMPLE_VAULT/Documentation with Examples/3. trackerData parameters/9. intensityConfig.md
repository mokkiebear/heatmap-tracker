# intensityConfig

`intensityConfig` is an object that allows you to configure the intensity scale and default values. It replaces the deprecated `defaultEntryIntensity`, `intensityScaleStart`, and `intensityScaleEnd`.

## Properties

- `scaleStart` (optional): The minimum value for the intensity scale.
- `scaleEnd` (optional): The maximum value for the intensity scale.
- `defaultIntensity` (required): The default intensity assigned to new data entries if no intensity is explicitly specified.
- `showOutOfRange` (required): Whether to show values that are outside the defined scale.
- `excludeFalsy` (optional): When set to `true`, entries with falsy intensity values (0, undefined, null, false) will be excluded from the heatmap. Default is `undefined` (falsy values are included).

## Example

```dataviewjs
const trackerData = {
    entries: [
        { date: "2025-01-01", intensity: 10 },
        { date: "2025-01-02", intensity: 1 },
    ],
    intensityConfig: {
        scaleStart: 1,
        scaleEnd: 10,
        excludeFalsy: true
        defaultIntensity: 5,
        showOutOfRange: true
    },
    heatmapTitle: "Heatmap with intensityConfig",
}

renderHeatmapTracker(this.container, trackerData);
```
