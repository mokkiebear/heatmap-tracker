# layout

> 📖 Full parameter reference: [README → `layout`](https://github.com/mokkiebear/heatmap-tracker#layout)

## Example

```heatmap-tracker
heatmapTitle: Monthly View
layout: monthly
property: exercise
```

## Combined with monthsToShow

```heatmap-tracker
heatmapTitle: Last 3 Months
layout: monthly
monthsToShow: 3
property: exercise
```
