# entries

`entries` is an array of objects representing the data points on the heatmap.

## Entry Properties

- `date` (required): The date of the entry in "YYYY-MM-DD" format.
- `intensity` (optional): The numeric value representing the intensity.
- `content` (optional): Text or HTML content to display in the tooltip or box.
- `customColor` (optional): A specific hex color for this entry, overriding the intensity color.
- `filePath` (optional): Absolute path to a file to open when clicked.
- `customHref` (optional): A custom URL to open when clicked (takes precedence over `filePath`).

## Example

```dataviewjs
const trackerData = {
    entries: [
        { 
            date: "2025-01-01", 
            intensity: 5, 
            content: "Happy New Year!" 
        },
        { 
            date: "2025-01-05", 
            intensity: 3, 
            customColor: "#ff0000", // Red color for this specific day
            content: "Important event"
        },
        {
            date: "2025-01-10",
            intensity: 8,
            customHref: "https://google.com", // Opens Google
            content: "Link to Google"
        }
    ],
    heatmapTitle: "Entries Example",
}

renderHeatmapTracker(this.container, trackerData);
```
