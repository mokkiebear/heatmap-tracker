Should be able to hide falsy values from the heatmap.

**Solution:**
- Add `excludeFalsy` property. `undefined` by default

```heatmap-tracker
heatmapTitle: test excludeFalsy ON
heatmapSubtitle: ""
property: exercise
year: 2025
separateMonths: false
showCurrentDayBorder: true
disableFileCreation: false
colorScheme:
  paletteName: default
ui: {}
intensityConfig:
    scaleStart: 0
    scaleEnd: 100
    defaultIntensity: 4
    showOutOfRange: true
    excludeFalsy: true
```
---

heatmapTitle: test excludeFalsy OFF
